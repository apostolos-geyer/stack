# docs

what's here and where to find it.

## the stack

- next.js app router
- prisma 7 with driver adapters (sqlite, postgres, supabase, neon, turso)
- better-auth
- trpc + react-query
- shadcn/ui + tailwind 4
- turborepo

## quick start

```bash
pnpm install
pnpm db:migrate:dev --name init
pnpm dev
```
